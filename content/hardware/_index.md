This section lists train controllers compatible with train simulation software and other software, grouped by type. Technical details are also provided, if they are known.

## Console controllers {#console}

Controllers for consoles which use a proprietary non-USB connector and usually pose as standard controllers from their corresponding console.

{{% hardware-list-type console %}}

## Game port controllers {#gameport}

Controllers for Microsoft Windows which use a [game port](https://en.wikipedia.org/wiki/Game_port).

{{% hardware-list-type gameport %}}

## Nintendo Wii controllers {#wii}

Controllers for the Nintendo Wii which use a Wii expansion cable and connect to a [Wiimote's extension port](https://wiibrew.org/wiki/Wiimote/Extension_Controllers).

{{% hardware-list-type wii %}}

## Serial port controllers {#serial}

Controllers for Microsoft Windows which use a DE-9 [serial port](https://en.wikipedia.org/wiki/Serial_port).

{{% hardware-list-type serial %}}

## Standalone controllers {#standalone}

Self-contained controllers with software installed in them. Instead of connecting to a PC or console, they can be connected directly to a TV or monitor via video output.

{{% hardware-list-type standalone %}}

## USB controllers {#usb}

Controllers which use a USB-A connector.

{{% hardware-list-type usb %}}
